# ATM Cash Demand Forecasting & Refill Route Optimization

## Overview

This project predicts ATM cash demand using Machine Learning to help banks optimize cash replenishment. The model analyzes historical ATM transaction data and forecasts future cash requirements, reducing cash shortages and unnecessary refill trips.

---

## Features

- ATM cash demand forecasting
- Data preprocessing
- Feature engineering
- XGBoost regression model
- Model performance evaluation
